O Html existe desde 199o cria pelo cientista Tim Berners-lee, e atualmente esta na versao 5, que veio recheada de recursos e funcionalidades que trazem melhorias para o Desenvolvimento Web

Sua Principal responsabilidade e dermacar a estrutura de uma pagina da web. Essa Estrutura do HTML e formada por um conjunto de elementos, ou seja, os hipertextos, que se conectam entre si formando a pagina. Os elementos sao utilizados para informar o navegador quer tipo de estrutura e essa que esta sendo construidam, podendo ser titulos, paragrafos, imagens, links, entre outros.

Dessa forma, para que um documento seja interpretado pelo navegador, e necessario que um arquivo tenha a extensao .html e a partir disso, podera ser exibido por qualquer navegador web.

Tags

As tags HTML sao elementos usados para marcar e formatar os conteudos: Textual ou visual em documentos HTML.

Uma tag HTML e uma marca que indica ao navegador que um determinado texto ou elemento deve ser interpretado conforme a regras do html

Elas sao formadas por uma abertura e uma fechadura, delimitadas por um simbolo de menor que e maior que

As tags HTMl basicas sao as tags que formam a estrutura basica de um documento,sao elas html,head,title,body,h1,p, etc. essas tag definem o conteudo e a formacao de um documento html.

elas sao usadas para estruturar o conteudo de um documento, para formatar o texto, para adicionar imagens e outros elementos, etc. As tags htmk bsucas sao essencias para criar qualquer documento html.
